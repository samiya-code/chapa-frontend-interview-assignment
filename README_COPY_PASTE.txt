QUICK START:
═════════════════════════════════════════════════════════════════════════════

1. Get Chapa API Key:
   → Visit https://chapa.co
   → Sign up (free)
   → Settings → API Keys → Copy Secret Key

2. Create .env file:
   CHAPA_SECRET_KEY=paste_your_key_here
   PORT=3001

3. Install dependencies:
   pnpm install

4. Run:
   pnpm dev

5. Open:
   http://localhost:3000

SUPPORT & TROUBLESHOOTING:
═════════════════════════════════════════════════════════════════════════════

Q: How do I get my Chapa API key?
A: Visit https://chapa.co → Sign up → Settings → API Keys

Q: Port 3000 is already in use?
A: Change port in vite.config.ts or kill the process using it

Q: Modules not found?
A: Run npm install or pnpm install again

Q: Server not starting?
A: Check if port 3001 is free, update .env file

Q: Payment not working?
A: Verify CHAPA_SECRET_KEY in .env is correct

Q: Want to customize colors?
A: Edit src/index.css and modify CSS variables
