GETTING STARTED:
═════════════════════════════════════════════════════════════════════════════

STEP 1: Download
   • Click the download button in v0 interface
   • Or extract the archive file provided

STEP 2: Extract
   tar -xzf chapa-dashboard-source.tar.gz
   cd v0-project

STEP 3: Get Chapa API Key
   • Visit: https://chapa.co
   • Sign up for free
   • Settings → API Keys → Copy Secret Key

STEP 4: Configure
   cp .env.example .env
   # Edit .env and add: CHAPA_SECRET_KEY=your_key

STEP 5: Install (if using source archive)
   npm install
   # or
   pnpm install

STEP 6: Run
   npm run dev
   # or
   pnpm dev

STEP 7: Open Browser
   http://localhost:3000

TROUBLESHOOTING QUICK REFERENCE:
═════════════════════════════════════════════════════════════════════════════

Issue: Cannot extract archive
  → Use 7-Zip (Windows), The Unarchiver (Mac), or tar (Linux)

Issue: npm install fails
  → Delete pnpm-lock.yaml or package-lock.json, try again
  → Use npm instead of pnpm

Issue: Port 3000 already in use
  → Kill process: lsof -ti:3000 | xargs kill -9
  → Or change port in vite.config.ts

Issue: Module not found
  → Delete node_modules folder
  → Run npm install again

Issue: CHAPA_SECRET_KEY not set
  → Create .env from .env.example
  → Add your Chapa API key

Issue: API calls failing
  → Verify API key is correct
  → Check internet connection
  → Review server logs
