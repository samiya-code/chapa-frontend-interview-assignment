## Quick Start (5 minutes)

### 1. Download & Extract
```bash
# Extract the recommended archive
tar -xzf chapa-dashboard-source.tar.gz
cd v0-project
```

### 2. Setup API Key
```bash
# Get your API key from https://chapa.co (free account)
cp .env.example .env

# Edit .env and add:
CHAPA_SECRET_KEY=your_key_here
```

### 3. Install & Run
```bash
npm install
npm run dev
```

### 4. Open Browser
```
http://localhost:3000
```
