## 🎯 Quick Start (After Extracting)

### **Step 1: Install Dependencies**
```bash
npm install
# or
pnpm install
```

### **Step 2: Setup Environment**
```bash
cp .env.example .env
```

Edit `.env` and add your Chapa API key:
```
CHAPA_SECRET_KEY=your_key_from_chapa.co
PORT=3001
```

### **Step 3: Run the Project**
```bash
npm run dev
# or
pnpm dev
```

### **Step 4: Open in Browser**
```
http://localhost:3000
```
